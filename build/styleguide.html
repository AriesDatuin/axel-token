                        Modal/Dialog Component

                        Modals can be used to load external HTML content within the same domain or subdomain. This component will not allow content loaded from other domains or websites for security and performance reasons. The component will only allow 1 modal at a time. Use modals sparignly!

                        Attributes

                        • data-content-url="path" - Url path to external content. Used on block-level elements such as div's and li's.
                        • data-content-selector="id-selector" - the selector id to load the content, ex: #my-selector
                        • data-content-width="400" - Define a fixed width or set to auto and the modal will resize appropriately according to the browser's width.
                        • data-content-height="600 - Define a fixed height or set to auto and the modal will resize appropriately according to the browser's height.
                        • modal-title="My Dialog Box" - For accessibility, use this class to define the content's main title.
                        • modal-description="This is a dialog box." - For accessibility, use this class to define the content's description.
                        Attributes

                        Used with a 'button' element, 'data-content-url' and sizing set to 'auto'.

                        data-content-url="path/to/modal/content" data-content-selector="id-selector" data-content-width="auto" data-content-height="auto" Copied!

                        Microinteractions (CSS Animations with JS Manipulation)

                        For microinteractions, the framework utilizes the Micron JS animation library. Microinteractions will trigger on mouseover as well as touchstart events.

                        Helpers

                        • data-micron="interaction" - Define an animtion type: blink, bounce, fade, flicker, groove, jelly, jerk, pop, shake, squeeze, swing, tada.
                        • data-micron-duration="number" - Define the duration (in seconds) of the animation.
                        • data-micron-timing="type" - Define the timing or easing: linear, ease-in, ease-out, ease-in-out.
                        • data-micron-bind="true" data-micron-id="id-selector" - Define the triggering element, and interaction will be applied to the element which reference to the defined id attribute.
