Modal Component

Modals can be used to load external HTML content within the same domain or subdomain. This component will not allow content loaded from other domains or web pages due to performance and security.

Attributes

  • data-content-url="path" - Url path to external content. Used on block-level elements such as div's and li's.
  • data-content-selector="id-selector" - the selector id to load the content, ex: #my-selector
  • data-content-width="400" - Define a fixed width or set to auto and the modal will resize appropriately according to the browser's width.
  • data-content-height="600 - Define a fixed height or set to auto and the modal will resize appropriately according to the browser's height.

Advanced Animations

By default animations are paused when the element is not in the viewport. When the element is visible in the viewport the animation will automatically play. Most animations will loop or repeat indefinitely. You can prevent this by using anim-interact and the element's animation will only play once when in the viewport and can play again on mouse or touch interaction.

Helpers

  • anim-play - Play the element's animation.
  • anim-interact - The element's animation will play once when in the viewport, and will play on mouse or touch interaction.
